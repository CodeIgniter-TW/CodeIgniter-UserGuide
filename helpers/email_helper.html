Email 輔助函數

Email輔助函數提供一些使用於電子郵件相關操作的函數。若要瞭解更多的電子郵件解決方案，請參閱 CodeIgniter 的 Email 類別。

載入輔助函數

使用下列的程式碼載入補助程式：

$this->load->helper('email');

下列為可以使用的函數：

valid_email('email')

檢查電子郵件是否為正確的電子郵件位址格式。請注意事實上並不表示這個位址一定能夠接收郵件，只是簡單地說明這是一個有效的電子郵件位址格式。

這個函數傳回 TRUE/FALSE。

$this->load->helper('email');

if (valid_email('email@somesite.com'))
{
    echo 'email is valid';
}
else
{
    echo 'email is not valid';
}

send_email('recipient'，'subject'，'message')

使用 PHP 內定的本機 mail () 函數發送一封電子郵件。如果需要更強大的電子郵件解決方案，請參考 CodeIgniter 的 Email 類。
