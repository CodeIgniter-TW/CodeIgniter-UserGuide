安裝指引

安裝 CodeIgniter 的四個步驟:

  1. 解壓縮
  2. 上傳 CodeIgniter 目錄及檔案到伺服器上。一般 index.php 檔案會在網頁的根目錄。
  3. 用編輯器開啟 application/config/config.php 設定 URL。如果你打算使用加密或是 session 的話，你可在這個檔案裡頭設定加密金鑰。
  4. 如果你打算使用資料庫的話，請開啟 application/config/database.php 填入你的資料庫資訊。

如果您希望隱藏 CodeIgniter 檔案位置來增加網站安全姓，您可以重新命名 system and application 資料夾，把它改成任何你想改的。如果已經修改過資料夾名稱，您必須編輯 index.php 檔案，將 $system_folder and $application_folder 這兩個變數換成您修改的目錄名稱。

最安全的做法是將 system 和任何 application 目錄移動到 Web 根目錄上一層，讓 Web 瀏覽器無法直接存取它。在預設狀況，每個目錄底下都包含 .htaccess 檔案，用來保護直接存取檔案，但是最好的方式就是全部移除 .htaccess，因為並非所有 Web 伺服器都支援 .htaccess 或者是 Web 伺服器改變設定。

如果移動了以上兩個資料夾，您必須編輯 index.php 檔案，並且將 $system_folder 和 $application_folder 變數，替換成絕對路徑，例如: '/www/MyUser/system'。

One additional measure to take in production environments is to disable PHP error reporting and any other development-only functionality. In CodeIgniter, this can be done by setting the ENVIRONMENT constant, which is more fully described on the security page.
