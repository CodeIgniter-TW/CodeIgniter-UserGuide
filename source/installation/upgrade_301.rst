#####################
從 3.0.0 升級到 3.0.1
#####################

在執行升級之前，你應該更換 index.php 為靜態檔案，將網站導向此靜態網頁。

步驟 1: 更新網站 CodeIgniter 檔案
=================================

取代 *system/* 目錄打下的所有目錄與檔案。

.. 注意:: 假如您有修改此目錄底下的檔案，麻煩請先複製備份。

步驟 2: 更新你的 CLI 錯誤訊息樣板
=================================

取代所有 *application/views/errors/cli/* 目錄底下的所有檔案。
